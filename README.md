# SOC Analyst Portfolio

> A professional portfolio documenting hands-on SOC analyst projects, network traffic analysis, Windows security, detection engineering, and Microsoft security technologies.

---

## Overview

This repository showcases practical work, technical documentation, and laboratory exercises focused on Security Operations Center (SOC) workflows.

The objective is to demonstrate hands-on experience in network analysis, Windows security, threat detection, SIEM technologies, and incident response while building a professional cybersecurity portfolio.

---

## Objectives

- Develop practical SOC Analyst skills
- Analyze network traffic and security events
- Investigate Windows Event Logs
- Perform packet analysis
- Create detection rules
- Write KQL queries
- Build Microsoft Sentinel experience
- Document incident response workflows
